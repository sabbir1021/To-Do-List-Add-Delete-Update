# Simple Todo App In Django

# Feature
	1. Create
	2. Edit
	3. Delete
	4. View

# Install Dependencies
```
pip install -r requirements.txt
```

# Procedure
	```
    python manage.py makemigrations
    python manage.py migrate
    python manage.py createsupersuer
    python manage.py runserver
    ```
